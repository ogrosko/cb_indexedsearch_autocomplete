Template
=========

Edit your template record and add cb_indexedsearch_autocomplete to the list “Include static (from extensions)”. This causes cb_indexedsearch_autocomplete to hook in every content page and makes cb_indexedsearch_autocomplete cooperate with other search frontends, like “macina searchbox”. If you just want to use cb_indexedsearch_autocomplete on the main search page you may skip this step.

Finally edit your template constants/setup regarding the following manual section.
